# 🐾 VetFlow AI - Automação Inteligente para Clínicas Veterinárias

O **VetFlow AI** é uma solução de automação Full-Stack projetada para otimizar o fluxo de agendamentos e a gestão de insumos em clínicas veterinárias. Este projeto demonstra como a Inteligência Artificial pode ser integrada a sistemas de inventário para reduzir perdas e aumentar o faturamento.

## 🚀 Funcionalidades Principais
*   **Agendamento Inteligente:** Chatbot capaz de interpretar solicitações de consulta via linguagem natural.
*   **Gestão de Estoque em Tempo Real:** Verificação automática de disponibilidade de vacinas e insumos antes da confirmação do agendamento (baseado na lógica do projeto FarmaStock).
*   **Prevenção de No-Show:** Sistema pronto para disparar lembretes automáticos.

## 🛠️ Tech Stack
*   **Backend:** Python (FastAPI)
*   **Database:** PostgreSQL (Modelagem relacional para agendamentos e inventário)
*   **IA:** Integração com APIs de processamento de linguagem natural.
*   **Arquitetura:** Clean Architecture com foco em escalabilidade.
